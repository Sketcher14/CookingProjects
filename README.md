# CookingProjects

A C++ learning and practice repository

## Overview

This project serves as a playground for exploring modern C++ features and implementing various algorithms and data structures from scratch.

## Subprojects

- **MultithreadingLab** - Concurrent programming experiments
- **Sandbox** - Experimental workspace for C++ practice
- **SortLab** - Algorithm implementations with comprehensive testing

## Requirements

- **C++ Compiler**: C++23 support required
- **CMake**: 3.26 or higher
- **Build System**: Ninja
